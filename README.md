# 🚀 DevGamificado

## 📖 Sobre o Projeto

O **DevGamificado** é uma plataforma web educacional criada para tornar o aprendizado de programação mais acessível, interativo e motivador para iniciantes.

O projeto foi desenvolvido como trabalho acadêmico da **A3 - Usabilidade, Desenvolvimento Web, Mobile e Jogos**, com foco na democratização do ensino de programação através da gamificação.

A plataforma oferece desafios práticos de programação, sistema de XP, conquistas e ranking competitivo, incentivando o aprendizado contínuo de forma semelhante aos jogos.

---

## 🎯 Objetivo

Desenvolver uma plataforma web gamificada que auxilie estudantes iniciantes no aprendizado de programação, reduzindo barreiras de entrada e promovendo inclusão digital.

## 🎮 Como Funciona

### 1. Painel Principal

O aluno visualiza:

* Nome do usuário
* Quantidade de XP
* Barra de progresso
* Conquistas desbloqueadas
* Lista de exercícios disponíveis

### 2. Exercícios

Os desafios estão organizados por linguagem:

* JavaScript
* Java
* C#

Cada exercício possui:

* Nível de dificuldade
* Quantidade de XP
* Correção automática

### 3. Sistema de Recompensas

| Conquista           | Requisito |
| ------------------- | --------- |
| 🏅 Primeiro Passo   | 10 XP     |
| ⚡ Código Veloz      | 40 XP     |
| 🧠 Mestre da Lógica | 80 XP     |
| 🚀 Dev Sênior       | 150 XP    |

### 4. Ranking

Os estudantes são classificados de acordo com a quantidade de XP acumulada, incentivando a competição saudável e a evolução contínua.

## 🌱 Impacto Social

O projeto está alinhado ao Objetivo de Desenvolvimento Sustentável (ODS 4) da ONU:

> Garantir educação inclusiva, equitativa e de qualidade para todos.

A proposta busca oferecer uma alternativa gratuita e acessível para o ensino de programação, especialmente para jovens estudantes e pessoas em transição de carreira.
